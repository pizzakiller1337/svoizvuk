Дипломный проект - сайт по продаже виниловых плстинок с функцией аудио-предпросмотра "Свой звук".
Работает на версии PHP 8.1 с базой данных My-SQL 8.0 на PhpMyAdmin
На сайте есть админ панель, она вынесена в отдельную папку admin
все картинки, аудиофайлы хранятся в папке assets
Сайт работает на хостинге Reg.ru ispmanager. Домен сайта - svoizvuk.online
В проекте лежит файл svoizvuk.sql в нем актуальная структура базы данных
